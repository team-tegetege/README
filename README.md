初期で嗜好も20程度の低次元のため、今回は独自の価値空間を内容ベース推薦システムに活用！  
![マッピングのコピー](https://user-images.githubusercontent.com/49345024/122665321-812d2b80-d1e1-11eb-8724-4997f289985c.jpg)
